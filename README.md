# Data and code for lithography technology evolution and forecasting

This repository provides the data and code supporting the manuscript:

**Dormancy and Technological Leap in Lithography: A Three-Dimensional Indicator Framework for Evolutionary Analysis and Forecasting**

## Overview

This study analyzes global granted lithography patents from 1918 to 2023 and develops an integrated framework combining Sentence-BERT-Kmeans semantic clustering, Logistic lifecycle identification, S–G–M technological theme classification, and ARIMA–LSTM hybrid forecasting.

## Repository structure

- `data/`: Aggregated and derived datasets used for lifecycle analysis, technological theme classification, and forecasting.
- `code/`: Python scripts for data preprocessing, theme identification, S–G–M classification, and forecasting.
- `outputs/`: Main empirical outputs, including theme classification results, forecasting results, and figure data.

## Data availability

The raw patent records were retrieved from the Incopat global patent database. Due to database licensing restrictions, the full raw patent records cannot be publicly redistributed. This repository provides aggregated and derived datasets, model inputs, empirical outputs, and analysis code sufficient to reproduce the main results reported in the manuscript.

The analysis was conducted using Python. Required packages are listed in `requirements.txt`.
